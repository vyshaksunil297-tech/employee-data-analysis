# Employee Data Analysis

## Overview

This project analyzes employee data to gain insights into workforce demographics, department distribution, education levels, work experience, locations, and salary trends. The analysis is performed using Python and data analysis libraries.

## Dataset Features

The dataset contains the following columns:

* Employee_ID
* Name
* Age
* Gender
* Department
* Job_Title
* Experience_Years
* Education_Level
* Location
* Salary

## Objectives

* Analyze employee demographics.
* Explore department-wise employee distribution.
* Examine salary trends across departments.
* Compare salaries based on education levels and experience.
* Identify workforce patterns by location and gender.
* Generate visualizations for better business insights.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

## How to Run

1. Clone the repository:
   git clone <repository-url>

2. Install required packages:
   pip install -r requirements.txt

3. Open the notebook:
   jupyter notebook

4. Run the analysis notebook.
